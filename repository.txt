HOW TO RUN LOCALLY

Prerequisites:
  - Python 3.11+
  - Node.js 18+
  - Git

Step 1 — Clone repositories:
  git clone https://github.com/ANSHRAJCODE/prepulse-backend
  git clone https://github.com/ANSHRAJCODE/prepulse-frontend

Step 2 — Backend setup:
  cd prepulse-backend
  python -m venv venv
  venv\Scripts\activate          (Windows)
  pip install -r requirements.txt
  Create .env file (see .env.example)
  python seed.py
  uvicorn main:app --reload

Step 3 — Frontend setup:
  cd prepulse-frontend
  npm install
  Create .env file with: VITE_API_URL=http://localhost:8000/api
  npm run dev

Step 4 — Open browser:
  http://localhost:5173
